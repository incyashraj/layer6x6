# Contributing to Layer36

Thanks for your interest. Layer36 is young and every contribution compounds.

## Where to start

1. Read the [Code of Conduct](https://github.com/incyashraj/layer6x6/blob/main/CODE_OF_CONDUCT.md). We enforce it.
2. Skim the [vision](../vision.md) and [roadmap](../roadmap.md).
3. Find a `good first issue`:
   <https://github.com/incyashraj/layer6x6/labels/good%20first%20issue>
4. Read [your first PR](first-pr.md) for a complete walkthrough.
