# Trademark Search: Layer36

**Status:** Preliminary search complete; lower apparent risk than former name  
**Owner:** Project maintainer  
**Last updated:** 2026-05-02

This file records the active Phase 0 name search for "Layer36". It is not legal
advice and does not replace review by a qualified trademark attorney.

## Summary

Layer36 is the selected project name as of 2026-05-02. A lightweight web search
did not surface an obvious exact-match software/runtime trademark conflict. Most
results for `Layer36` or `Layer 36` appear to be incidental references to a
numbered layer in products, patents, 3D printing logs, model layers, profile
names, or fiction chapters.

Layer36 is still not legally cleared. Before broad public launch, complete the
official trademark searches and decide whether to file or continue under common
law use.

plans, docs, draft launch copy, CLI placeholders, WIT examples, and bundle naming.

Before Phase 0 exit:

- Complete official USPTO search for `Layer36`, `Layer 36`, and close variants.
- Complete WIPO, EUIPO, J-PlatPat, CNIPA, and India searches.
- Search domain and package registry availability.
