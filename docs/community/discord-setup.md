# Discord Setup Checklist

Phase 0 calls for a small, useful community server. Keep it quiet and technical
until Layer36's official name search and docs site are ready.

## Channels

- `#welcome`
- `#announcements`
- `#general`
- `#dev`
- `#rfc`
- `#help`
- `#off-topic`

## Welcome Message

```text
Welcome. This is the early community for Layer36, a universal app platform for
native cross-platform software.

Start here:
- Read the roadmap and Phase 0 status.
- Ask setup questions in #help.
- Keep technical decisions in GitHub issues, PRs, discussions, or ADRs.
- Be kind, precise, and patient with new contributors.
```

## Rules

- Follow the repository Code of Conduct.
- No harassment, spam, or personal attacks.
- Keep project decisions in durable GitHub records.
- Do not present Layer36 as a cleared trademark until official search is complete.

## Launch Order

1. Complete official name search or explicitly accept the naming risk.
2. Publish GitHub Pages.
3. Create Discord server and pinned welcome/rules.
4. Add invite link to README and docs.
5. Invite the first five trusted contributors.
