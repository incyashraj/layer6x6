# Code of Conduct

This policy is adapted from the structure and enforcement expectations of
Contributor Covenant 2.1.

OneOS is a technical community for building a universal application platform.
We want it to be rigorous, welcoming, and worth people's time.

## Expected behavior

- Be respectful and constructive, especially when disagreeing.
- Assume good intent first, then ask clarifying questions.
- Keep feedback focused on ideas, code, docs, and process.
- Make room for newer contributors and answer basic questions generously.
- Respect boundaries, privacy, and different communication styles.

## Unacceptable behavior

- Harassment, threats, stalking, or sustained disruption.
- Personal attacks, insults, or demeaning language.
- Sexualized comments or unwelcome sexual attention.
- Publishing private information without explicit permission.
- Repeatedly pushing a discussion after maintainers ask for a pause.

## Reporting

Report concerns to `conduct@layer6x6.dev`.

If that address is not yet active during Phase 0, contact the repository owner
privately through GitHub. We will acknowledge reports within 72 hours whenever
possible, handle them with discretion, and prioritize contributor safety.

## Enforcement

Maintainers may remove comments, lock threads, reject contributions, suspend
community access, or ban participants when needed. Enforcement decisions should
be proportional, documented privately, and revisited if new information appears.

This policy applies in the repository, issue tracker, pull requests, official
chat spaces, events, and any other project-controlled venue.
